How to handle 13 machine columns?

13 columns won't fit on screen at once. How should we organize them?

A

Group by Type (Collapsible)

Group machines into categories that expand/collapse. Your boss sees 5-6 groups instead of 13 columns, and expands the group he's working with.

B

Filter Tabs + Horizontal Scroll

Tab bar at top: "All Machines" | "Cutting" | "Forming" | "Roll Forming" | "CNC/Wood" | "Finishing". Each tab shows only those machine columns. "All" shows everything with horizontal scroll.

C

Horizontal Rows (Swim Lanes as Rows)

Flip it: each machine is a horizontal row, jobs are cards going left-to-right in priority order. All 13 machines visible vertically — scroll down instead of sideways. Drag cards left/right to reorder, drag up/down to move between machines.
